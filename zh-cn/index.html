SiaChart - 快速交易图表

源码 文档 博客

  • 高效快速

    基于Skia 2D（用于Chrome/Android图形/Flutter），运行稳定快速。

  • 跨平台

    桌面端：Win/Mac/Linux，移动端：Android/iOS

  • 集成方便

    可配置编译为静态/动态库，提供C/C++ API接口，方便对接各种开发语言。

  • 脚本支持

    内置Lua虚拟机，可使用脚本扩展图表功能。支持自设计语法解析器。

  • 指标公式系统

    集成Ta-lib，支持百种常用指标。统一开发接口，方便自定制开发公式指标。

  • 可定制化

    外观/功能可定制，清晰的开发接口，定制开发为任意类型交易图表。

  • git checkout https://github.com/qiuchengw/siachart.git
  • cd siachart & mkdir build & cd build
  • cmake -A x64 ..
